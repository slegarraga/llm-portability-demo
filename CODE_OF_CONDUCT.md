# Code of Conduct

## Expected behavior

Be respectful, direct, and constructive. Focus criticism on code, examples,
documentation, and maintainer workflow.

## Unacceptable behavior

Harassment, threats, discriminatory language, sexualized content, doxxing, and
repeated bad-faith disruption are not acceptable.

## Enforcement

Maintainers may edit, hide, or remove comments; close issues or pull requests;
block users; or report abuse to GitHub when needed to keep the project usable.

Report conduct issues privately to `sebastian@0a.cl`.
