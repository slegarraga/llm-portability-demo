# Contributing

Thanks for helping improve the portability demo.

## Local setup

```sh
npm install
npm start
```

The demo runs offline against canned provider data, so contributions should not
require API keys.

## Pull requests

- Keep changes focused on provider-portability examples.
- Preserve the zero-runtime-dependency packages used by the demo.
- Update `README.md` when behavior or output changes.
- Run `npm start` before opening a pull request.

## Issues

For bugs, include the command you ran, the observed output, and the expected
portable provider behavior.
